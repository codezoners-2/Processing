functions: the rules

  • we must pass the same number of parameters as declared in the definition of the function
  • the parameters must be the same type as in the definition of the function (int to int, float to float, etc)
  • we can give a value, a variable or even the result of a fuction to another function
  • variables declared within a function are known only locally
  • functions receive copies of the variables not the variables themselves

fractals

  • from Latin “fractus” (broken)
    • shapes that repeat on many levels
  • these shapes don't even have to be the same on the different levels, as long as they share some similarity features
  • they exist everywhere in nature and they are a much more accurate way of describing many of it's phenomena
