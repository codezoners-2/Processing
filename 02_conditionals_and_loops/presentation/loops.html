conditionalAnimation / whileExample

loops - while( )

  • As long as the sentence inside parentheses is true the loop is going to run
  • As soon as the sentence is false, it's going to exit the curly brakets and run the code under the while block

Attention: if the sentence is always true, the program is going to get stuck

loops - for( )


void draw()
{
// INITIALIZATION, EVALUATION AND UPDATE ALL IN ONE LINE

  for (int circleX=10; circleX < width; circleX = circleX + 30)
  {
    ellipse(circleX, circleY, 20, 20);
  }
}

forExample

loops - for( )

  • updating the variable happens at the end of each cycle
  • the variable that is initialized is local (it's known only within the loop, we can't use it outside)
  • we can build a loop within loop if we want to add extra dimensions

for (int x = 0; x < 10; x++)
{
   for (int y = 0; y < 10; y++)
   {
     point( x, y);
   }
}
